编程常用快捷键

Posted on 2020-07-08 |

像我本人这样的新手在写代码的过程中，会出现频繁使用鼠标的情况，这经常让人感觉到不适，特别是要多次调试代码的时候。除此之外，编码过程中使用“↑↓←→”移动光标改写代码时十分费力。而且对于代码强迫症患者来说，代码不对齐简直是天大的灾难。上述几种情况都十分浪费宝贵的编码时间。

之前看到大佬们写代码用快捷键，编码速度快到飞起，我不禁心生羡慕。这是我在网上搜索并摘录快捷键设置(有些快捷键仅适用于visual studio code)。

当然我列出的仅包含我觉得比较常用的一些快捷键，等熟悉之后会增添一些。一些简单的没有列出(比如 Ctrl + a / c / v / z：全选/复制/粘贴/撤销)。

快捷键 作用
Ctrl + N 新建文件
Ctrl + Tab 文件之间切换
Ctrl + b 开/关侧边栏
Ctrl + F 查找
Ctrl + H 查找替换
F11 全屏显示(再次按则恢复)
Ctrl + [ / ] 代码行向左或向右缩进
Ctrl + / 单行注释
Ctrl + L 选中当前行
Ctrl + X 删除行
Ctrl + Enter 在前一行代码的任何位置调到下一行
Home(Fn + ←) / End(Fn + →) 移动到行首/行尾
Alt + ↑ / ↓ 向上/向下移动行
Shift + Alt + A 多行注释
Alt + ↑ / ↓ 向上或向下移动一行
Shift + Alt + F 代码格式化
Shift + Alt + ↓ / ↑ 向上/向下复制行
