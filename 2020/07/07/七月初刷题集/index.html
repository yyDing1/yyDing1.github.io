Codeforces 1367D

题目难度：1800

题目大意

给定一个字符串$s$和一个数组$a_1, a_2, …, a_n$，通过删除其中一些字符，并随机打乱顺序，使对于任意$1 \leq i \leq n$满足以下条件：

解题思路

首先，对于答案中最大的字符，所有其他字符都比它要小(或等)，所以对应的$a$数组中此位置必定是0。运用贪心策略，找出$a[i]$中为0的位置填入字母，每次都填最大的，但是要事先检验能填进去的字符数要比此时数组$a$中0的个数要大。填入以后消除填入字符的位置对于其他位置的影响。由此，填长度为$n$的串被简化成填长度为$n - 1$的串…以此类推，可得出最终答案。

Codeforces 1369D

难度：1900

题目大意

先定义了一种RDB树：起始只有一个节点。每次扩展操作：遍历所有存在的节点：

  1. 如果该节点有3个孩子，不进行操作。
  2. 如果该节点有1个孩子，该节点增添两个孩子。
  3. 如果该节点没有孩子，该节点增添两个节点。

下图是一个层数为4的RDB树

然后定义了claw树：一个父亲节点带三个孩子节点

问：高度为n的RDB树可以对多少个节点均未被染色的claw子树进行染色。

解题思路

不难发现，对于level = n的RDB树来说，他是由两个level = n - 2的RDB树和一个level = n - 1的RDB树构成的。

所以这是一个动态规划题。但是这并不是简单的$dp[i] = dp[i - 1] + dp[i - 2]$，构成level = n的RDB树还要对level = n - 2的RDB树和一个level = n - 1的RDB树的顶端染色进行讨论。所以设对顶端未染色的最大染色claw树数目为$dp[i][0]$, 已染色的为$dp[i][1]$，所以可以得到以下状态转移关系：

  • $dp[i][0] = max(dp[i - 1][0], dp[i - 1][1]) + 2 \times max(dp[i - 2][0], dp[i - 2][1])$
  • $dp[i][1] = dp[i - 1][0] + 2 \times dp[i - 2][0] + 1$

官方题解通过对顶点取法具体证明得出了一个更加普遍的结论：

  • $dp[i] = 2 \times dp[i - 2] + dp[i - 1] + (imod3 == 0? 1:0)$

$dp[i]$表示level = i的RDB树最大染色的claw子树个数

Codeforces 1368D

难度：1700

题目大意

给你一个数组$a_1, a_2, …, a_n$，可以进行的操作：取出两个数$x, y$，计算出$xANDy$和$xORy$，再放进原数组里面去。

问：经过有限次操作后$\sum_{i = 1}^{n}{a_i^2}$的最大值是多少。

解题思路

首先$xANDy + xORy = x + y$，那么对于数组无论进行什么操作，$\sum_{i = 1}^{n}{a_i}$始终不变。要使得$\sum_{i = 1}^{n}{a_i^2}$，就要使数组中的数两极分化到极限。所以策略是每轮操作都变换出一个最大的数。

$xORy \geq x, y$，所以$a_1|a_2|…|a_n$就是最大的数(设为新的$a_1$)，此时的$a_1|$任何数都是它本身，故最大。将$a_1$取出，对剩下n-1个数进行相同操作。依次类推。$O(n^2)$。将全部操作转换为数位操作，先全部进行2进制数位统计再去取，复杂度可简化为$O(nk)$，本题的$k = 20$。

总结

总之，经过这个阶段的刷题我得出的经验是：

  • 不要乱下结论，不要乱下结论，不要乱下结论，不要相信自身经验性或者样例表征性的结论，要一步一步去证明，每一步都不要疏漏。
  • 不要过于急功近利。不要怕麻烦，要多编样例和反例找漏洞，不要急着提交。如此急着提交只会增加wa掉以后的挫败感(十分自闭)。
  • (Last but not least)心态要好，不要因为wa or tle而自闭，自闭也不要自闭很久，毕竟很浪费时间。
