基本描述

对于一个字符串$S$，它对应的后缀自动机是一个最小的确定有限状态自动机（DFA），接受且只接受$S$的后缀。
DFA：DFA可以用一个五元组 <字符集，状态集，转移函数、起始状态、终结状态集>来表示。

状态分类的方法(原则)：把S的所有子串的$endpos$都求出来。如果两个子串的$endpos$相等，就把这两个子串归为一类。

学习过程中一个错误的理解：只会$len(substrings) = 1$，$endpos > 1$ 或$len(substrings) > 1$，$endpos = 1$的情况，也就是说只需合成一次。这种想法是错误的，反例为$abcabc$，要严格按照上述分状态的方法，事实上有些状态是要合成很多次的。

一些性质

  • $s_1$是$s_2$的后缀当且仅当$endpos(s_1) \supseteq endpos(s_2)$，$s_1$不是$s_2$的后缀当且仅当$endpos(s_1) \cap endpos(s_2) = \phi$
  • 对于一个状态$st$，以及任意$s\in substrings(st)$，都有$s$是$longest(st)$的后缀
  • 对于一个状态$st$，以及任意的$longest(st)$的后缀$s$，如果$s$的长度满足：$length(shortest(st)) <= length(s) <= length(longsest(st))$，那么$s\in substrings(st)$，也就是说$substrings(st)$包含的是$longest(st)$的一系列连续后缀。

SAM的后缀链接

$aabbab$,$abbab$,$bbab$,$bab$的$endpos$都是${6}$，下一个$ab$当然也在结束位置$6$出现过，但是$ab$还在结束位置$3$出现过，所以$ab$比$aabbab$,$abbab$,$bbab$,$bab$出现次数更多，于是就被分配到一个新的状态中了。

我们可以发现一条状态序列：$7\rightarrow 8\rightarrow 5\rightarrow S$。这个序列的意义是$longest(7)$即$aabbab$的后缀依次在状态$7,8,5,S$中。我们用后缀链接$SuffixLink$这一串状态链接起来，这条$link$就是上图中的绿色虚线。

transition function

我们将$st$遇到的下一个字符集合记作$next(st)$，有$next(st) = \{S[i+1] | i \in endpos(st)\}$，例如$next(S)=\{S[1], S[2], S[3], S[4], S[5], S[6], S[7]\}=\{a, b, d\}$，$next(8)=\{S[4], S[7]\}=\{b, d\}$

对于一个状态$st$和一个字符$c\in next(st)$，可以定义转移函数$trans(st, c) = \{x | longest(st) + c \in substrings(x)\}$
