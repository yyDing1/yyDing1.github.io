poj2778的升级版
感觉是个神题

题目大意

给定一些词根，计算长度L以内至少含有一个词根的单词有多少个

解题思路

  • 考虑不含这些词的单词总数。
  • AC自动机建图，将每个单词结尾的节点标记, 将标记点排除在外。
  • 由trie图建出邻接矩阵mat，每走一步相当于乘一个trie图的邻接矩阵。
  • 出发地为i，s步之后到达j的路线总数为$mat^{s}[i][j]$。
  • 显然只需统计$\sum_{i = 0}^{dimension}{nowmat[0][i]}$, 但是很显然复杂度会爆炸。
  • 解决方案: 将矩阵扩展一维, 将最后一列全部设为1, 这样要求的答案就是$mat^{L + 1}[0][dimension + 1]$, 这样就可以采用矩阵快速幂解决。
  • 相同的, 求$26 + 26^2 + 26^3 + … + 26^n$也可以通过快速幂求得。
  • 两者相减, 即为答案。
  • mod 2^64 只需开unsigned long long 自然溢出
