赛后补题

1011.Kidnapper’s Matching Problem

  • 线性基 + kmp

题意

给定一个长度为n的数组$a$，一个长度为m的模式数组$b$以及另一个数组$S$，定义

用模式数组$b$去匹配$a$(即对于所有的$l \in[1, n - m + 1]$，匹配$[b_1, b_2, …, b_m]$和$[a_l, a_{l + 1}, …, a_{l + m - 1}]$)，匹配成功的条件为:

求：
$\sum_{i = 1}^{n - m + 1}[(a_1, a_2, …, a_{i + m - 1})\quad matches\quad b]\times 2^{i - 1}\quad mod\quad (10^9 + 7)$

关于线性基

个人理解线性基类似于质因数分解中的数与每个素数的关系，每个素数互不相交并且能表示任意一个数。

线性基具有的性质：

  1. 每个基的最高位均不相同
  2. 基数之间不能互相表示，即线性基中不存在$a\bigoplus b=c$
  3. 线性基的数量是一定的，但是值不是一定的
  4. 求解线性基的思路类似于贪心的思想

解题思路

求出线性基，用线性基对$a$和$b$的影响，接下来问题就变成了模式串匹配问题，跑一遍$kmp$即可。
