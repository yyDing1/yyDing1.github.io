  • 因需考虑硬币数量，所以每次状态转移要记录硬币数量。

HDU1176

假设成每一个数据(x, t)都对应x-t坐标系上的一个点
对于其中的任意两个点a(x1, t1)可到达b(x2, t2)的条件为

  • t2 > t1且(t2 - t1) / abs(x2 - x1) >= 1
  • 斜率不存在
  • 两点重合
    以上三种情况可以归结为：
  • x2 + t2 >= x1 + t1且t2 - x2 >= t1 - x1
    问题转化为找满足上式的最多的点
    储存每个点的值arr[i].first = t + x, arr[i].second = t - x
    以arr[i].first为第一关键字, arr[i].second为第二关键字升序排序
    找所有的arr[i].second组成的数组的最长上升子序列(包含相等)
