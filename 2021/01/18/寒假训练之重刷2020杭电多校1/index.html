确认过眼神，都是我不会做的题

Cookies

缩略版题意：求$f(i) = \sum \limits_{j = 0}^{i}{divmed(j)}$，$divmed(j)$表示$j$的所有$d$个因子的从小到大的第$\lceil \frac{d}{2}\rceil$个数，单次访问求$f(n)$，$n <= 10^{10}$

第一次看题解写的是分段打表，还特地表明要打$10$分钟，$wtf?$

开一个$4000$的数组，每$2500000$个数记录一个值，求值只需要先找到跟要找的数最近的下标，再从$\leq 1250000$的区间中求这个区间的贡献。

$divmed(i)$等价表示为$\leq \sqrt i$的最大因子。

对于一段区间$[l, r]$，$\sum \limits_{j = l}^{r}{divmed(j)}$可用$[1, \sqrt r]$中的数通过若干次覆盖得到，覆盖次数为$(r - l + 1) \sum \limits_{i = 1}^{\sqrt r}{\frac{1}{i}} \approx (r - l + 1)ln(\sqrt r)$

复杂度为$1250000ln(\sqrt{10^{10}}) \approx 1.4 \times 10^{7}$，河里

如果没有记错，我的表打了二十多分钟$qwq$

Distinct Sub-palindromes

签到题

题意：给定长度，找循环子串最少的串的个数

  • $n > 3$，循环子串最小为$3$，且是$1,2,3,1,2,3,…,1,2,3$的形式，答案为$C_{26}^{3}$
  • $n \leq 3$枚举情况，得到$n$是任意取都可，答案为$A_{26}^{n}$

Fibonacci Sum

卡常头被卡没了

题意：给定$n, c, k$求$\sum \limits_{i = 0}^{n}{(F_{ic})^k}$，$F$是斐波那契数列。$1\leq n,c \leq 10^{18},1\leq K\leq 10^{5}$.

根据$F[i] = F[i - 1] + F[i - 2]$这一性质可求得$F$的通项公式：

$F[i] = \frac{1}{\sqrt 5}((\frac{1 + \sqrt 5}{2})^{i} - (\frac{1 - \sqrt 5}{2})^{i})$

令$d = \frac{1}{\sqrt 5},a = \frac{1 + \sqrt 5}{2},b = \frac{1 - \sqrt 5}{2}$

$F[i] = d(a^{i} - b^{i})$

$\sum \limits_{i = 0}^{n}{(F_{ic})^k}=\sum\limits_{i = 0}^{n}{(d(a^{ci} - b^{ci}))^k}$

$= d^{k}\sum\limits_{i = 0}^{n}{(a^{ci} - b^{ci}})^k$

$=d^{k}\sum\limits_{i = 0}^{n}\sum\limits_{j = 0}^{k}{C_{k}^{j}(a^{ci})^{j}(-b^{ci})^{k - j}}$

$=d^{k}\sum\limits_{j = 0}^{k}\sum\limits_{i = 0}^{n}{C_{k}^{j}(a^{cj})^{i}(-1)^{k - j}(b^{c(k - j)})^{i}}$

$=d^{k}\sum\limits_{j = 0}^{k}C_{k}^{j}(-1)^{k - j}\sum\limits_{i = 0}^{n}{(a^{cj}b^{c(k - j)})^{i}}$

$=d^{k}\sum\limits_{j = 0}^{k}C_{k}^{j}(-1)^{k - j} \frac{(a^{cj}b^{c(k - j)})^{n + 1} - 1}{a^{cj}b^{c(k - j)} - 1}$

如果$a^{cj}b^{c(k - j)} = 1$，求和里面取$C_{k}^{j}(-1)^{k - j}(n + 1)$

对于每一个$j$都可以$O(log(mod))$求出，总复杂度$O(klog(mod))$

接下来就是卡常了$qwq$

  • 无理数是可以写成逆元形式的，$\sqrt 5$的逆元是$383008016$($383008016 \times 383008016 \%1000000009 = 5$)
  • 初始化的时候只求阶乘和阶乘逆元就行了，求多了常数会变大
  • 求阶乘逆元可以线性求，先求出最大的数的逆元再往前推
  • 先欧拉降幂，再快速幂
  • 对于像$a^{cj}$和$b^{c(j - k)}$这样的量可以动态维护，没必要每次都求一遍，减少了循环里快速幂的次数
  • 最最最最重要的一点：$a^{cj}b^{c(k - j)} = 1$的情况要特判$qwq$

Minimum Index

加深了对$Lyndon$分解的理解，记录一下心路历程

题意：给定一个字符串$s$，找出每个前缀的最小后缀

  • 后缀排序可以求$s$的最小后缀，但不适用于$s$的前缀
  • 考虑$Lyndon$分解，每次加入一个字符
  • 对于一个字符串，他的最小后缀为最后一个$Lyndon$分解的最后一个$Lyndon$串
  • 考虑每次加入一个字符
  • 如果当前恰好能形成一个$Lyndon$分解，那么显然当前答案就是最后一个$Lyndon$串
  • 否则，当前状态为一些$Lyndon$串和一个伪$Lyndon$串，答案一定在最后一个伪$Lyndon$串中得到
  • 原本的一个思路是：答案为最后一个字符所在周期字符串，经验证发现是错的。因为$Lyndon$串的一个前缀未必是$Lyndon$串，如$aab\ aab$，对此段进行$Lyndon$分解时判断第五个$a$时会把其最小后缀判为$aa$，这显然是不对的。
  • 题解提出了一个做法是：维护伪$Lyndon$串的最小公共前后缀，这是正确的，因为$Lyndon$串$s$某个前缀$t$的后缀$p$小于$s$的情况就是$p$既是$t$的一个后缀也是$t$的一个前缀。但是维护当前伪$Lyndon$串的最小公共前后缀有些麻烦，没想出什么好办法。
  • 网上的题解提出了一个很好的解决办法，上述情况可以归结为：在考虑更大周期的情况时会忽略更小周期所对应的情况。而第一个周期求出的答案一定是正确的，这就类似于一个递归的思想
    • 若当前周期$T$大于$1$，则当前答案$beginpos[now] = beginpos[now - T] + T$
    • 若当前周期$T$等于$1$，则当前答案即为当前周期串
    • 模拟一下上面的错误，确实这样是可以避免的
    • 比如上面的$aabaab$，第$1$个$aab$对应的答案是对的
