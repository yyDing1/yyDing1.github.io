Example

жүҫеҲ°зҡ„д»ҘдёӢеҚЎзү№е…°ж•°зҡ„й—®йўҳпјҡ

  1. жңүеӨҡе°‘з§ҚдёҚеҗҢзҡ„й•ҝеәҰдёә$2n$зҡ„жӢ¬еҸ·еәҸеҲ—пјҹ
  2. жңү$2n$дёӘдәәжҺ’жҲҗдёҖиЎҢиҝӣе…Ҙеү§еңәгҖӮе…Ҙеңәиҙ№$5$е…ғгҖӮе…¶дёӯеҸӘжңү$n$дёӘдәәжңүдёҖеј $5$е…ғй’һзҘЁпјҢеҸҰеӨ–$n$дәәеҸӘжңү$10$е…ғй’һзҘЁпјҢеү§йҷўж— е…¶е®ғй’һзҘЁпјҢй—®жңүеӨҡе°‘дёӯж–№жі•дҪҝеҫ—еҸӘиҰҒжңү$10$е…ғзҡ„дәәд№°зҘЁпјҢе”®зҘЁеӨ„е°ұжңү$5$е…ғзҡ„й’һзҘЁжүҫйӣ¶пјҹ
  3. дёҖдҪҚеӨ§еҹҺеёӮзҡ„еҫӢеёҲеңЁеҘ№дҪҸжүҖд»ҘеҢ—$n$дёӘиЎ—еҢәе’Ңд»Ҙдёң$n$дёӘиЎ—еҢәеӨ„е·ҘдҪңгҖӮжҜҸеӨ©еҘ№иө°$2n$дёӘиЎ—еҢәеҺ»дёҠзҸӯгҖӮеҰӮжһңд»–д»ҺдёҚз©ҝи¶ҠпјҲдҪҶеҸҜд»Ҙзў°еҲ°пјүд»Һе®¶еҲ°еҠһе…¬е®Өзҡ„еҜ№и§’зәҝпјҢйӮЈд№ҲжңүеӨҡе°‘жқЎеҸҜиғҪзҡ„йҒ“и·Ҝпјҹ
  4. еңЁеңҶдёҠйҖүжӢ©$2n$дёӘзӮ№пјҢе°ҶиҝҷдәӣзӮ№жҲҗеҜ№иҝһжҺҘиө·жқҘдҪҝеҫ—жүҖеҫ—еҲ°зҡ„$n$жқЎзәҝж®өдёҚзӣёдәӨзҡ„ж–№жі•ж•°пјҹ
  5. еҜ№и§’зәҝдёҚзӣёдәӨзҡ„жғ…еҶөдёӢпјҢе°ҶдёҖдёӘеҮёеӨҡиҫ№еҪўеҢәеҹҹеҲҶжҲҗдёүи§’еҪўеҢәеҹҹзҡ„ж–№жі•ж•°пјҹ
  6. дёҖдёӘж ҲпјҲж— з©·еӨ§пјүзҡ„иҝӣж ҲеәҸеҲ—дёә$1,2,3,..,n$жңүеӨҡе°‘дёӘдёҚеҗҢзҡ„еҮәж ҲеәҸеҲ—пјҹ
  7. $n$дёӘз»“зӮ№еҸҜжһ„йҖ еӨҡе°‘дёӘдёҚеҗҢзҡ„дәҢеҸүж ‘пјҹ
  8. $n$дёӘ$+1$е’Ң$n$дёӘ$-1$жһ„жҲҗ$2n$йЎ№$a_{1},a_{2},вҖҰ,a_{2n}$пјҢе…¶йғЁеҲҶе’Ңж»Ўи¶і$a_{1} + a_{2} + вҖҰ + a_{k} \geq 0(k = 1,2,3,вҖҰ,2n)$еҜ№дёҺ$n$иҜҘж•°еҲ—дёәпјҹ
  9. зҺ°еңЁжңү$2n$дёӘдәәпјҢд»–д»¬иә«й«ҳдә’дёҚзӣёеҗҢпјҢд»–д»¬иҰҒжҲҗдёӨжҺ’пјҢжҜҸдёҖжҺ’жңү$n$дёӘдәәпјҢе№¶дё”ж»Ўи¶іжҜҸдёҖжҺ’еҝ…йЎ»жҳҜд»Һзҹ®еҲ°й«ҳпјҢдё”еҗҺдёҖжҺ’зҡ„дәәиҰҒжҜ”еүҚдёҖжҺ’еҜ№еә”зҡ„дәәиҰҒй«ҳпјҢй—®жңүеӨҡе°‘з§ҚжҺ’жі•?

д»ҘдёҠеҪўжҲҗзҡ„ж•°еҲ—йғҪжҳҜеҚЎзү№е…°ж•°гҖӮ

Definition

е…¶йҖ’жҺЁе…ізі»пјҲд№ҹдҪҝе…¶еҺҹе§Ӣе®ҡд№үпјүдёә$H_{n} = \sum \limits_{i = 0}^{n}{H_{i}H_{n - 1 - i}}$ $(*)$

Some Proof

$(*) \Leftrightarrow(1)$пјҡ$H_{n} = \sum \limits_{i = 0}^{n}{H_{i}H_{n - 1 - i}}$еҫҲеҘҪзҗҶи§Ј

$(2) \Leftrightarrow(1)$пјҡ$5$е…ғзӣёеҪ“дәҺ$($пјҢ$10$е…ғзӣёеҪ“дәҺ$)$

$(3) \Leftrightarrow(1)$пјҡеҗ‘еҸізӣёеҪ“дәҺ$($пјҢеҗ‘дёҠзӣёеҪ“дәҺ$)$

$(4) \Leftrightarrow(1)$пјҡ$H_{n} = \sum \limits_{i = 0}^{n}{H_{i}H_{n - 1 - i}}$еҫҲеҘҪзҗҶи§Ј

$(5) \Leftrightarrow(4)$пјҡжҳҫз„¶

$(6) \Leftrightarrow(1)$пјҡеҮәж ҲзӣёеҪ“дәҺ$($пјҢе…Ҙж ҲзӣёеҪ“дәҺ$)$

$(7) \Leftrightarrow(1)$пјҡ$H_{n} = \sum \limits_{i = 0}^{n}{H_{i}H_{n - 1 - i}}$еҫҲеҘҪзҗҶи§Ј$or$еҗ‘дёӢйҒҚеҺҶзӣёеҪ“дәҺ$($пјҢеҗ‘дёҠеӣһжәҜзӣёеҪ“дәҺ$)$

$(8) \Leftrightarrow(1)$пјҡ$+1$е…ғзӣёеҪ“дәҺ$($пјҢ$-1$е…ғзӣёеҪ“дәҺ$)$

$(9) \Leftrightarrow(1)$пјҡзҹ®йҳҹдәәзӣёеҪ“дәҺ$($пјҢй«ҳйҳҹдәәе…ғзӣёеҪ“дәҺ$)$

еӨ§йғЁеҲҶй—®йўҳйғҪеҸҜд»ҘиҪ¬жҚўжҲҗеҺҹе§Ӣе®ҡд№үжҲ–жӢ¬еҸ·й—®йўҳ

йҖҡйЎ№е…¬ејҸ

еҜ№дәҺ$(2)$пјҢжҠҪиұЎдёә$(0, 0) \rightarrow (n, n)$дё”д»»ж„Ҹж—¶еҲ»$y \leq x$зҡ„ж–№жЎҲж•°

еҸ–еҸҚпјҡ$(0, 0) \rightarrow (n, n)$дё”еӯҳеңЁж—¶еҲ»$y \geq x + 1$зҡ„ж–№жЎҲж•°пјҢе°Ҷ$(n, n)$е…ідәҺ$y = x + 1$еҜ№з§°$(n - 1, n + 1)$пјҢеҚідёә$(0, 0) \rightarrow (n - 1, n + 1)$зҡ„ж–№жЎҲж•°пјҢдёә$2n \choose n - 1$

жҖ»ж–№жЎҲж•°дёә${2n \choose n} - {2n \choose n - 1} = \frac{1}{n + 1} {2n \choose n}$

еҚЎзү№е…°ж•°зҡ„йҖҡйЎ№е…¬ејҸдёә$H_{n} = \frac{1}{n + 1} {2n \choose n}$

з”ҹжҲҗеҮҪж•°

$G(x) = \sum\limits_{i = 0}^{\infty}{H_{i}x^i}$пјҢжұӮ$H_{i}$

$H_{n} = \sum \limits_{i = 0}^{n}{H_{i}H_{n - 1 - i}}$зұ»дјјдәҺдёӨдёӘ$G(x)$зҡ„еҚ·з§Ҝ

$G(x)\times G(x) = \sum\limits_{i = 0}^{\infty}{x^{i}\sum\limits_{j = 0}^{i}{H_{j}H_{i - j}}}=\sum\limits_{i = 0}^{\infty}{H_{i + 1}x^{i}}$

$xG^2{(x)}=\sum\limits_{i = 0}^{\infty}{H_{i + 1}x^{i + 1}} = 1 + G(x)$

$xG^2{(x)} = 1 + G(x)$

$G(x) = \frac{1\pm\sqrt{1-4x}}{2x}$

еҜ№$\sqrt{1-4x}$иҝӣиЎҢеҲҶи§Ј

$(1 - 4x)^{\frac{1}{2} } = \sum\limits_{i = 0}^{\infty}{ {\frac{1}{2} \choose i}{(- 4x)^{i} } }$

$=\sum\limits_{i = 0}^{\infty}{\frac{(\frac{1}{2})(\frac{1}{2} - 1)(\frac{1}{2} - 2)вҖҰ(\frac{1}{2} - i + 1)}{i!}}(-4x)^{i}$

$=\sum\limits_{i = 0}^{\infty}{\frac{-(-\frac{1}{2})^{i}\times 1\times3\times5\timesвҖҰ\times(2i - 3)}{i!}}(-4x)^{i}$

$=\sum\limits_{i = 0}^{\infty}{-\frac{1\times3\times5\timesвҖҰ\times(2i - 3)}{i!}}(2x)^{i}$

$=\sum\limits_{i = 0}^{\infty}{-\frac{1\times3\times5\timesвҖҰ\times(2i - 3)}{i!}}\times \frac{2\times4\times6\timesвҖҰ\times(2i)}{1\times2\times3\timesвҖҰ\times i}x^{i}$

$=\sum\limits_{i = 0}^{\infty}{- \frac{(2i)!}{(2i - 1)(i!)^2}}x^{i}$

$=\sum\limits_{i = 0}^{\infty}{-\frac{1}{2i - 1}{2i \choose i}}x^{i}$

жҳҫз„¶$G(x)$еҸ–$\frac{1 -\sqrt{1-4x}}{2x}$

$G(x) = \frac{1 + \sum\limits_{i = 0}^{\infty}{\frac{1}{2i - 1}{2i \choose i}}x^{i}}{2x}$

$=\sum\limits_{i = 0}^{\infty}{\frac{1}{2(2i - 1)}{2i \choose i}}x^{i - 1}$

$=\sum\limits_{i = 0}^{\infty}{\frac{1}{2(2i + 1)}{2i + 2 \choose i + 1}}x^{i}$

$=\sum\limits_{i = 0}^{\infty}{\frac{1}{2(2i + 1)}\frac{(2i + 2)(2i + 1)}{(i + 1)^2}{2i \choose i}}x^{i}$

$=\sum\limits_{i = 0}^{\infty}{\frac{1}{i + 1}{2i \choose i}}x^{i}$

жүҖд»Ҙ$H_{n} = \frac{1}{n + 1}{2n \choose n}$
